DevOps is a methodology/culture/mindset in the software development world which Involves set ofpractices to bridge the gap of Dev and ops team by using Open source automation build tools.
Devops Automation:Automation is the process of reducing human interaction with IT systems by creating scripts or functions that are repeatable and can be used either by other software or on command.
Devops scaling:Scaling in DevOps is an automation-driven environment where developers or enterprises can adopt in DevOps culture and choose the right technologies to scale(update,upgrade or adopt) based on the business demand.
Devops infrastucture:Idea of Managing infrastructure with the help of code which is done with the help of certain tools or programs which can help to carry out the tasks automatically.
Importance of devops:DevOps is important because it's a software development and operations approach that enables faster development of new products and easier maintenance of existing deployments.Its provides product with better 
quality and efficiency by adopting new ideas and its impletation faster.
